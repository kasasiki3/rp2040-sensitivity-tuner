How to Use
  1. Click Connect and select your device ※ Web Serial API requires Chrome or Edge
  2. Click Send to write default settings, or Request to load current settings from the device
  3. Adjust sensitivity and delay values with the sliders
  4. Click Send to upload the new settings to the device
  5. Click Save if you want to persist the settings Settings are written to the device's flash memory

Threshold value for each sensor. A key press is triggered when the sensor voltage exceeds this value. Lower values = higher sensitivity. If you experience double-hits, raise the sensitivity (increase the value) for the affected pad.

Adelay Default: 1 ms

Lockout time after a key press before the same key can fire again. Increase this if you get double-hits on the same pad only.

Bdelay Default: 12 ms

Minimum interval between consecutive Don (center) hits. If set too high, fast rolls on the center will be missed.

Cdelay Default: 30 ms

Same as Bdelay but for Rim (edge) hits.

Ddelay Default: 32 ms

Suppression time between a Don hit and a Rim hit (and vice versa). Increase this if hitting the center also triggers the rim, or the other way around.

Hdelay Default: 20 or 0 ms

Input rate limiter for official apps and some simulators. Use this if input works in Notepad but not in a taiko simulator.

How to Tune Sensitivity

Repeat steps 1 → 3, aiming for the highest sensitivity with the lowest delay possible.

  1. Set sensitivity Start from the default values and find the minimum threshold at which every hit registers reliably.
  2. Adjust delays Raise each delay until double-hits stop. Note: values above 35 ms tend to cause roll detection failures.
  3. Fine-tune sensitivity If double-hits remain after delay tuning, lower sensitivity (raise the value) for the affected pad.
Firmware Update
Prep Remove the screws from the connector and take out the board
1 Connect to PC in DFU mode

Hold the BOOT button on the microcontroller board and plug in the USB cable. If Explorer shows a new storage drive, the connection was successful.

2 Download the firmware from GitHub

① Click the Code button  →  ② Select Download ZIP and save the file.

Extract the downloaded ZIP archive.

3 Flash the .uf2 file

Find HIDtaiko.uf2 in the extracted folder and drag & drop it onto the DFU storage drive. When the drive disconnects automatically, flashing is complete.
